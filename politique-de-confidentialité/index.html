Player Analytics

Dernière mise à jour : 15 août 2021

Ce serveur utilise Plan, une plateforme d'analyse des joueurs.

Les données recueillies sont stockées localement sur le serveur et ne sont pas transférées à des tiers.

S'il y a une violation des données, le propriétaire du serveur devra en informer les utilisateurs.

Le transfert de ces données à un tiers nécessitera le consentement de toutes les parties concernées, à moins que les données utilisateur pseudonymisées soient supprimées/anonymisées avant le transfert.

Ce que Player Analytics recueille à votre sujet (lié à l'UUID) :

  • Votre UUID (identifiant unique) et votre nom de joueur.
  • Pays d'où vous vous êtes connecté
  • Noms d'affichage utilisés (tels que ceux modifiés avec /nick )
  • Date de votre première visite sur le serveur
  • Sessions
  • Dates d'entrée et de sortie lorsque vous avez été sur le serveur pour calculer le temps de jeu.
  • Informations sur le temps passé dans chaque monde et mode de jeu.
  • Informations sur le temps passé en AFK
  • Combien de monstres ont été tués
  • Combien de décès ont eu lieu
  • Informations sur le ping (délai par rapport au serveur)
  • Les meurtres de joueurs (liés à l'utilisateur), l'heure et le nom de l'arme.

Ce que Player Analytics recueille sur vous (anonyme) :

  • Joueurs en ligne chaque minute
  • Combien de fois chaque commande est utilisée
  • Combien d'entités et de chunks sont chargés sur le serveur.

Comment ces données sont-elles stockées ?

Les données sont stockées dans une base de données SQL dans un format pseudonymisé.

  • Pour relier les données à une personne réelle, il faut une clé dans ce cas, l'UUID et l'accès aux bases de données des comptes de Mojang.

(Ces bases de données ne sont pas accessibles au public).

Comment ces données sont-elles utilisées ?

Ces données sont utilisées pour fournir au personnel des informations analytiques sur leur serveur.

Afin qu'ils puissent prendre des décisions plus éclairées pour améliorer leur serveur.

Les informations suivantes sont calculées à partir des données disponibles :

  • La "santé" du serveur
  • Les joueurs, ont-ils des gens avec qui jouer ?
  • Les joueurs, restent-ils sur le serveur ?
  • Combien de nouveaux joueurs restent des joueurs réguliers ?
  • Combien de joueurs uniques restent sur le serveur chaque jour ?
  • Quelles sont les heures les plus actives sur le serveur ?
  • Quel est le monde le plus joué, et dans les réseaux, quel est le serveur le plus joué ?

Vos droits :

  • Droit de demander la suppression de vos données.
  • Droit d'être informé des données que nous détenons sur vous (accès à votre propre page de joueur dans la configuration par défaut du plan).
