Tableau #

Commandes Description Disponible pour
/spawn Vous téléporte au spawn du serveur après 5 secondes. Utilisable une fois toutes les 15 minutes Invité, Membre, Donateur
/home Vous téléporte à votre domicile après 5 secondes. Utilisable une fois toutes les 15 minutes Donateur
/lock Verrouille une porte, un coffre ou un four pour que vous seul puissiez y accéder. Vous pouvez avoir jusqu'à 20 blocs verrouillés Membre, Donateur
/f Affiche les commandes des factions Membre, Donateur
/ticket Crée une demande de ticket pour que le personnel réponde Membre, Donateur
/sethome Définit l'emplacement de votre domicile, revenez-y avec la commande /home Membre, Donateur
/help Affiche toutes les commandes que vous pouvez utiliser Invité, Membre, Donateur
/tpa [Nom d'utilisateur] Demandes de téléportation vers un autre joueur Donateur
/tpahere Nom d'utilisateur] Demandes à un autre joueur de se téléporter vers vous Donateur
/money Affiche le nombre de jetons que vous possédez actuellement Invité, Membre, Donateur
